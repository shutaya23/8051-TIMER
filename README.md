# 8051-TIMER
1系統名稱:多功能計時器 
ACS - Alarm Clock Simulation  
2 系統背景:
8-bit 8051 microcontroller unit (MCU)  

利用8051單晶片結合4x4矩陣鍵盤、蜂鳴器，實作出可模擬多功能的計時器


3 基本功能:
計時(或計數)、七段顯示器顯示、鍵盤輸入、中斷控制、led閃爍、蜂鳴器發出特定頻率、lcd螢幕顯示、七段顯示器開始到數、中斷按鈕


4 模擬應用功能
LCD顯示器顯示:Welcome timer simulation、
計時到0蜂鳴器發出鈴響且led燈閃爍、
中斷回主選單、
設定現在時間、
顯示當前時間、
LCD字幕提示、
LCD螢幕消除、
顯示特定訊息
